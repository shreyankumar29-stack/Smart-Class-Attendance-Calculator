# Project Title

Smart Class Attendance Calculator

## Problem Statement

Students often find it difficult to manually track attendance across multiple courses and determine whether they meet the minimum attendance requirement (e.g., 75%). Calculating attendance percentages and predicting how many classes can be missed without falling below the required threshold is time-consuming and prone to errors.

## Proposed Solution

Develop a web-based Class Attendance Calculator that enables students to record attendance for each class, automatically calculate subject-wise attendance percentages, predict the number of classes they can safely miss while maintaining the required attendance, and notify them when their attendance is close to falling below the minimum requirement.

## Key Features

* Add a class (Attended or Not Attended)
* View current attendance percentage for each course
* "How many classes can I miss and still maintain 75% attendance?" calculator
* Attendance warning (e.g., "You're at 74%. One more absence and you'll fall below the required attendance.")
* Display total classes held and total classes attended for every course
* Simple dashboard to monitor attendance across all subjects

## Technical Approach

* Frontend: HTML5, CSS3, JavaScript, Bootstrap
* Backend: Python Flask
* Database: SQLite
* Charts: Chart.js
* Version Control: Git & GitHub
* Deployment: Render

## Success Metrics

* Accurate attendance percentage calculations
* Correct safe bunk prediction based on attendance target
* Real-time attendance warnings for low attendance
* User-friendly and responsive interface
* Successful deployment with complete GitHub documentation
